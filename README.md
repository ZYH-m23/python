# python
#1.面向过程与面向对象
#面向过程：分析解决问题的步骤，把步骤拆解成一个个方法，没有对象调用的，通过一个个的方法去执行（是指用手洗衣服）
#面向对象：将编程当作一个事物，直接使用，不用管过程（是指用洗衣机洗衣服）
#2.类与对象：
#类：一系列具有相同属性与行为的事物的统称，不是真实存在的事物
#对象是类的具体体现，是类创建出来的真实存在的事物，面向对象的核心
#在开发中，先有类，再有对象
#2.1类的三要素
#1.类名
#2.属性：对象的特征描述，用来说明是什么样子
#3.方法：对象具有的功能，用来说明能做什么
#2.2定义类
#基本格式
class类名:#类名符合标识符规定，并采取大驼峰命名法
